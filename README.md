# wb_super_sign
**这是一个油猴脚本。**

功能：一键签到你关注的所有微博超级话题。

注意：只能在你的个人主页上操作。签到后打开控制台查看结果。

安装地址：https://greasyfork.org/zh-CN/scripts/26293
